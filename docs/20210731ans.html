Question 2

Comment whether each of the following is true.

  1. Aqueous chlorine can decolorize acidified potassium permanganate solution.

    2. F: both are oxidizing agents!

  2. A compound cannot be an oxidizing agent and a reducing agent at the same time.

    3. F: disproportionation reaction.

  3. Free electrons can travel through an electrolyte solution.

    4. F: Mobile ions can travel through an electrolyte solution.

  4. Weak oxidizing agents are strong reducing agents.

    5. F: very common wrong concept! Check the ECS again.

  5. Sulfuric acid ionizes completely in water.

    6. F: the second hydrogen atom does not ionize completely.

  6. An acid with a higher basicity is more basic.

    7. F: basicity refers to the number of potentially ionizable hydrogen atoms in a molecule.
